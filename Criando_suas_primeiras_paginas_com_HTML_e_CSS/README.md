CSS:
- Os seletores podem ser por: tag, id (#), classe (.), universal (*) e atributo;
- Combinadores:
    | Combinador | Símbolo |
    |------------|----------|
    | descendente| [espaço]|
    | filho      |    >    |
    | irmão adjacente | +  |
    | irmão em geral |  ~  |
- Cores:
   - Pré-definidas;
   - Current-color;
   - Hexadecimal (#0000FF);
   - Hexadecimal com transparência (#000000AA);
   - RGB (rgb(255, 0, 0));
   - RGBA rgba(0, 0, 0, .8);
   - HSL hsl(360, 100%, 50%);
   - HSLA hsla(140, 100%, 50%, .8)
